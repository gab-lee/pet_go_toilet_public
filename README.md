# pet_go_toilet_public
**Description**
- Pet go toilet is a telegram bot designed in partnership witha SG small business [Fuji Haven](https://fujihaven.com) to help pet owners track the poop and pee of their pets. 

Click [Telegrame Bot](https://t.me/petpoopoobot) to access the telegram bot. 

## Features
- Add and manage pets 
|-- Users may rename/remove pets if needed.
- Add poop/pee events
|-- Users can specify the time/location/success of the event
- Summary of events
